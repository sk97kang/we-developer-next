## Crawling

정보수집의 효율성을 위해 python으로 crawling 한다.

1. 원하는 사이트 주소를 입력한다.
2. 해당 사이트에 `meta`tag에서 title, descript 등 정보를 추출한다.
3. 정보를 json 형식으로 저장한다.
4. 가져와서 사용한다.

> Cawler : https://repl.it/@sk97kang/metacrawler#main.py
